# 모심로봇(MOSIM) — 로봇 간병인 서비스 기획서

## 1. 문제 정의

한국은 2025년 만 65세 이상 인구가 **1,084만 명(총인구의 약 20%)**에 달해 초고령사회에 진입했다(통계청, 2025). 동시에 WHO는 2030년 전 세계 **의료 인력 약 1,000만 명 부족**을 예고했고, 국내 가정 간병비는 **월 200~300만 원대**로 가구에 직접적인 부담이 되고 있다(업계 통계).

그러나 현존하는 가정용 AI 돌봄 제품은 대부분 **감지·대화·알림 중심**이다. 효돌·다솜은 정서 대화와 복약 리마인드에 특화돼 있고, 삼성 Ballie·LG Q9 역시 홈 IoT 대화 에이전트 성격이 강하다. **기립 보조·보행 지원·재활 동작 교정 같은 물리적 개입**이 가능한 가정 제품은 여전히 공백 상태이며, 이 간극이 "알림은 있으되 위기에 몸을 쓸 수 없다"는 구조적 문제를 만든다.

### 주제 선택 이유

- AI Robotics 도메인 중 **초고령사회 대응**은 사회적 시급성과 시장 증거(월 200~300만 원 간병비)가 동시에 명확하다.
- VLA(Vision-Language-Action) 등 최신 로봇 AI가 연구·시연 단계를 벗어나 **실제 운용 단계로 진입**하고 있다. Foxconn·NVIDIA **Nurabot**은 대만 병원 실증에서 간호 업무 **20~30% 절감**을 보고했고(CNN 2025-09-12), 국내 **K-휴머노이드 연합**은 2028년 상용화에 1조 원+ 투자를 발표했다(2025-04). 다만 이 흐름은 **병원·기관 중심**이며, **가정 케어는 여전히 공백**이다. 본 서비스는 이 공백을 선점하는 **가정 응용 계층**으로 기획 가치가 높다.

## 2. 타겟 사용자

| 계층 | 정의 | 핵심 니즈 |
|---|---|---|
| **1차** | 독거노인 (65+ 1인 가구) | 응급 조기 감지, 낙상 대응, 정서적 동반, 복약 관리 |
| **2차** | 수술 후 회복 환자 (퇴원 ~ 4주) | 병원→가정 연속 케어, 보행 재개 보조, 이상징후 원격 공유 |
| **3차** | 외래 재활 환자 | 처방 운동 동작 교정, 세션 간 가정 내 반복 훈련 보조 |

공통 니즈는 **"사람이 곁에 없는 시간의 물리·인지·의료 공백"**이다. 이는 기존 알림형 기기가 해결하지 못하는 축이다. 구매 의사결정은 주로 **자녀 세대(40~60대)**가 하므로, 브랜드 축은 효(孝) 정서와 기술 신뢰를 함께 설계한다.

## 3. 서비스 개요

**서비스명: 모심로봇 (MOSIM)** — 한국어 "모시다"의 어간에서 왔으며, 효 문화에 부합하는 브랜드 정체성을 전달한다.

**핵심 기능 4가지**

1. **거동 보조** — 기립·착석·단거리 보행 시 손잡이·리프팅 지원
2. **이상징후 감지·전파** — 실시간 보행 패턴·안색·호흡 베이스라인을 개인별로 학습하고, 이탈 시 **보호자 앱·주치의·응급센터**에 자동 알림
3. **복약 확인** — 시간·약봉지·복용 여부를 영상으로 확인, 미복용 시 리마인드 후 보호자 통지
4. **재활 동작 교정** — 물리치료사가 처방한 운동을 시연·관찰하고 자세·횟수에 피드백

**기존 서비스 대비 차별점**

| 축 | 기존 (효돌·다솜·Ballie·Q9) | 모심로봇 |
|---|---|---|
| 개입 방식 | 알림·대화·정서 | **물리 보조 + 다모달 인지** |
| 데이터 관점 | 단발 이벤트 | **개인화 베이스라인** 시계열 |
| 외부 연계 | 가전·앱 | **의료진·응급망 양방향** |

## 4. AI 기술 적용

| 기능 | 적용 기술 | 선택 이유 |
|---|---|---|
| 지시 이해·행동 실행 | **VLA (Vision-Language-Action)** | 영상·음성 상황을 단일 정책으로 행동까지 매핑. π0.5·GR00T N1·Gemini Robotics·RoboNurse-VLA 등은 **현재 연구·시연 단계**이며, 본 서비스는 공개 모델을 **가정 케어 태스크로 파인튜닝**하여 활용 |
| 이상징후 감지 | 개인화 시계열 이상탐지 (LSTM / Transformer 계열) | 보행 속도·보폭·안색·호흡의 **개인 베이스라인 이탈**을 탐지해 일반 임계치 대비 오탐을 낮춤 |
| 노인 발화 인식 | 온디바이스 경량 STT (방언·저음·느린 발화 특화) | 네트워크 단절 시에도 응급 명령 인식 가능, 개인정보 보호 |
| 비접촉 생체 감지 | mmWave 레이더 호흡·심박 | 카메라 사각·야간에도 동작, 수면 중 이상 감지에 유리 |

## 5. 시스템 구성

```
[가정 — 로봇 본체]

  ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
  │     센서     │ ──▶ │   엣지 AI    │ ──▶ │  액추에이션    │
  │    RGB-D     │     │   VLA 추론   │     │   거동 보조    │
  │ mmWave 레이더 │     │     STT      │     │   동작 교정   │
  │    마이크     │     │   이상탐지    │     │              │
  └──────────────┘     └──────┬───────┘     └──────────────┘
                              │ 암호화 업로드
                              ▼
                      ┌──────────────────┐
                      │     클라우드      │
                      │  개인 베이스라인   │
                      │   학습·대시보드    │
                      └─────────┬────────┘
                                │
            ┌───────────────────┼───────────────────┐
            ▼                   ▼                   ▼
     ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
     │  보호자 앱    │    │ 주치의·의료진  │    │  응급 콜센터   │
     └──────────────┘    └──────────────┘    └──────────────┘
```

- **엣지에서 1차 추론·응급 판단** → 네트워크 단절·지연 시에도 핵심 안전 기능 유지
- **클라우드는 베이스라인 학습·장기 모니터링·대시보드** (익명화·암호화)
- **알림은 보호자·주치의·응급 3채널**로 동시 전송해 누락 방지

## 6. 기대 효과

**사용자 관점**
- 응급 감지부터 고지까지 시간 단축 → 골든타임 내 조치 확률 상승
- 기립·보행·복약 등 일상을 독립적으로 수행 → **자립 생활 연장**
- 가족이 실시간 상태를 확인 → 심리·물리적 돌봄 부담 경감

**비즈니스 관점**
- 가정 간병 시장(월 200~300만 원)의 **부분 대체** 포지션. 구독형(RaaS) 진입으로 초기 도입 장벽 완화 — IFR World Robotics 2025에서 서비스로봇 RaaS **+31%** 성장이 확인됨
- 조기 감지에 따른 **의료비 절감** 효과 → 향후 보험 수가 편입 시 B2G 확장 여지
- **K-휴머노이드 연합(2028 목표)** 정책 흐름과 합치 → 공공·민간 파트너십 확보에 유리

## 7. 향후 발전 방향

| Phase | 내용 |
|---|---|
| **1 (1~2년)** | **가정에 들어가는 첫 버전** — 거실 고정형 기기 + 바퀴형 로봇, 다모달 센서로 조기 감지·3채널 자동 알림·물건 전달 (월 구독) |
| **2 (3~5년)** | **걸어다니는 돌봄 로봇** — 경량 휴머노이드가 기립·이송·계단 보조, 지자체 시범사업 참여 |
| **3 (5년+)** | **병원·가정 연속 케어** — 건강보험 수가 적용 원격 모니터링, 해외 진출 |

기술 로드맵: 온디바이스 VLA 경량화 → 노인 특화 STT 데이터셋 확보 → 햅틱·안전 인증(KC/KS) → 의료기기 소프트웨어(SaMD) 허가 연계.
